loco.Manager

Definition via Loco.lization Manager


var manager = new loco.Manager(); manager.for("en-US").area("dashboard").text("welcome").is("Welcome to $0, we ${hope|really hope}$ $1 $[have|has]$ a good time here!");

Usage via Loco.lization Manager


manager.for("en-US").area("dashboard").text("welcome").with('Loco manager', 'your multilingual team').opt('{','}',0).sing
